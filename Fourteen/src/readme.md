# Reflection 三种类型

- Class.forName(“全类名”)
  - 
- 类名.class
  - 作为参数传递
- 对象.getClass
  - 已经创建了对象后，才能使用

# Java Class 三个阶段

-  源代码阶段
  - .java -> .class , Class.forName("全类名")

- 加载内存阶段
  - .class -> memory loading , 类名.class
  
- 对象运行阶段
  - Class a = new Class() , 对象.getClass()

# 全类名

  包名 + 类名 
  通过快捷键的方式拷贝、粘贴，避免出错。
